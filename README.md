# AiFusion

## 平台简介

### 模块

| 项目                    | 说明                 |
|-----------------------|--------------------|
| `ai-fusion-dependencies`  | Maven 依赖版本管理       |
| `ai-fusion-framework`     | Java 框架拓展          |
| `ai-fusion-server`        | 管理后台 + 用户 APP 的服务端 |
| `ai-fusion-module-system` | 系统功能的 Module 模块    |
| `ai-fusion-module-member` | 会员中心的 Module 模块    |
| `ai-fusion-module-infra`  | 基础设施的 Module 模块    |
| `ai-fusion-module-bpm`    | 工作流程的 Module 模块    |
| `ai-fusion-module-pay`    | 支付系统的 Module 模块    |
| `ai-fusion-module-mall`   | 商城系统的 Module 模块    |
| `ai-fusion-module-erp`    | ERP 系统的 Module 模块  |
| `ai-fusion-module-crm`    | CRM 系统的 Module 模块  |
| `ai-fusion-module-mp`     | 微信公众号的 Module 模块   |
| `ai-fusion-module-report` | 大屏报表 Module 模块     |
